# XEMU-Main-Files For Windows 

- Go to **[Latest Release](https://github.com/Poop-fish/XEMU-Main-Files/releases/tag/Xemu)** and download the Zip file  
- Extract Zip  
- Start up Xemu  
- Load the files  
- Restart then Load your Game
